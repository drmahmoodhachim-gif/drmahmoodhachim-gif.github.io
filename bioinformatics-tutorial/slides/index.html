What Is RNA-seq?

RNA sequencing measures which genes are expressed (turned on) in your sample and how much.

  • More reads = more mRNA = gene is more active
  • We compare counts between conditions (e.g., healthy vs diseased)
  • Goal: find genes that change significantly

Experimental Design

  • Define your groups (Control vs Treatment)
  • Use biological replicates (at least 3–4 per group)
  • Keep groups balanced
  • Randomize to avoid batch effects

DESeq2 – Differential Expression

Statistical test: Which genes change between conditions?

  • log2 Fold Change – how much (positive = up, negative = down)
  • padj – adjusted p-value (< 0.05 = significant)
  • DEG – differentially expressed gene

Volcano Plot

X = fold change | Y = significance

Red points = significant DEGs (padj < 0.05, |log2FC| > 1)

Upper right = upregulated | Upper left = downregulated

MA Plot

X = average expression | Y = fold change

Used to check for bias – most points near zero = good

Heatmap

Top DEGs × samples. Red = high, blue = low.

Shows which samples cluster together and which genes change together.
