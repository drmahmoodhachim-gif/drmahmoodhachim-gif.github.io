  1. Download the tutorial → Download ZIP
  2. Extract the ZIP file. Open the bioinformatics-tutorial folder inside.
  3. Install R from cran.r-project.org
  4. Run by double-clicking RUN_ANALYSIS.bat

You only need R. No Python, no command line experience required.

Analysis Steps – What Each Step Means

StepWhat It DoesWhy It Matters
1. Count matrixTable of reads per gene per sample (rows=genes, columns=samples)Raw expression data – higher counts = more mRNA = gene more active
2. DESeq2Compares Control vs Treatment, tests which genes differGives fold change, p-value, padj – identifies significant DEGs statistically
3. Volcano plotX=fold change, Y=significance (−log10 padj)Quick view of which genes are large and significant (red = DEGs)
4. MA plotX=mean expression, Y=fold changeChecks for bias – ensures low-count genes aren’t spurious
5. HeatmapTop DEGs × samples, color = expression levelShows patterns – which samples cluster, which genes change together
6. Pathway barplotDEGs grouped by biological pathwayMoves from genes to biology – which processes are affected

Key terms: log2FC = fold change (positive = up, negative = down); padj = adjusted p-value (<0.05 = significant); DEG = differentially expressed gene.
