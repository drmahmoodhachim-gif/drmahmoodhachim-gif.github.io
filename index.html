RNA-seq Pipeline Tutorial

DEG Analysis & Systems Biology – For 7 PhD Students | No Python Required

Quick Start

  1. Download the tutorial → Download ZIP
  2. Extract the ZIP file. Open the bioinformatics-tutorial folder inside.
  3. Install R from cran.r-project.org
  4. Run by double-clicking RUN_ANALYSIS.bat
  5. View results in 05_deg_results/volcano_plot.png

You only need R. No Python, no command line experience required.
